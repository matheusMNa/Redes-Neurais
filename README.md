# Redes-Neurais - PINNs

Esse repositório tem como objetivo explorar PINNs - Physics-Informed Neural Networks, rede neurais que seguem um paradigma que incorpora leis físicas na forma de equação diferenciais no cáulculo de resíduos e no processo de otimização. O arquivo principal é um Jupyter Notebook que, além de explicar com certa profundidade o que são as PINNs, faz dois casos de aplicação em Python com dados sintéticos: resolve um problema direto na forma de aproximar as soluções da EDO do decaimento exponencial e um problema inverso na forma de descobrir o valor de frequência em um oscilador harmônico.
